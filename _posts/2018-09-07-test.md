#  Vue.js  #
是一款用于构建用户界面的渐进式框架，Vue 被设计为可以自底向上逐层应用。

## 1.如何导入Vue  ##
    <!-- 开发环境版本，包含了有帮助的命令行警告 -->
    <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>

    <!-- 生产环境版本，优化了尺寸和速度 -->
    <script src="https://cdn.jsdelivr.net/npm/vue"></script>


## 2.声明式渲染 ##
Vue.js 的核心是一个允许采用简洁的模板语法来声明式地将数据渲染进 DOM 的系统<br>
  
    <div id="app">
    	{{ message }}
    </div>

    <script>
    	var app = new Vue({
    	  el: '#app',
    	  data: {
    	message: 'Hello Vue!'
    	  }
    	})
    </script>
